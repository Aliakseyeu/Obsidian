
// RefreshDatabase:
1. Пересоздает всю БД из миграций
2. Таблицы полностью пустые
3. Нет конфликтов с существующими данными
4. Автоинкременты сбрасываются

1. Работает с существующей БД
2. Удаляет данные только в транзакции
3. UNIQUE constraints видят "старые" данные
4. Автоинкременты НЕ сбрасываются
