
```mysql
AVG(TIMESTAMPDIFF(YEAR, user_birthday, CURDATE()))
```

Этот SQL-запрос вычисляет **средний возраст пользователей** на основе их дней рождения.

|                                 |                                           |
| ------------------------------- | ----------------------------------------- |
| `CURDATE()`                     | Текущая дата                              |
| `user_birthday`                 | Дата рождения пользователя                |
| `TIMESTAMPDIFF(YEAR, ..., ...)` | Разница в полных годах между двумя датами |
