
```
project::\$types[^\s]*\[0\]
```

- `project::` — буквально
- `\$types` — экранированный `$`
- `[^\s]*` — **любой символ, кроме пробела**, 0 или больше раз
- `\[0\]` — литеральное `[0]`

---
## Что он найдёт ✅

```
project::$types[0] 
project::$types->foo[0] 
project::$types['bar'][0] 
project::$typesXYZ[0]
```

---

## Что он НЕ найдёт ❌

```
project::$types [0]      // пробел 
project::$types ->foo[0] // пробел
```

---

## Если нужно минимум один символ между

(то есть `$types[0]` не подходит):

```
project::\$types[^\s]+\[0\]
```
