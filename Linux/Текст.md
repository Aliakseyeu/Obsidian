
Переписать все содержимое файла

```bash
echo "text" > filename.txt
```

---

Дописать в конец файла

```bash
echo "text" >> filename.txt
```

---

Дописать в начало файла

```bash
echo "Новая строка" | cat - filename.txt > temp && mv temp filename.txt
```

```bash
sed -i '1i Новая строка' filename.txt
```

---

Вывести все содержимое файла

```bash
cat filename.txt
```

---

Вывести первые 10 строк или 5

```bash
head filename.txt
head -n 5 filename.txt
```

---

Вывести последние 10 строк или 5

```bash
tail filename.txt
tail -n 5 filename.txt
```

---

Вывести начало и конец

```bash
(head -n 5 filename.txt; echo "..."; tail -n 5 filename.txt)
```
