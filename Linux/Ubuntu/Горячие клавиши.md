```bash
gsettings list-recursively org.gnome.desktop.wm.keybindings | grep ...
```

---

### Установить горячую кнопку

```bash
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-right "['Page_Up']"
```

---

### Удалить горячую кнопку

Чтобы убрать конкретное сочетание, сбрось его в пустой список:

```bash
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-left "[]" gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-right "[]"
```

После изменения комбинация должна перестать работать сразу (иногда нужно перезапустить сессию или разлогиниться).
