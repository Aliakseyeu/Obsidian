В GNOME все [[Горячие клавиши]] для рабочих столов лежат тут:

```bash
gsettings list-recursively org.gnome.desktop.wm.keybindings | grep workspace
```

Там будут строки типа:

```bash
org.gnome.desktop.wm.keybindings switch-to-workspace-left   ['<Super>Page_Up'] org.gnome.desktop.wm.keybindings switch-to-workspace-right  ['<Super>Page_Down'] org.gnome.desktop.wm.keybindings switch-to-workspace-1      ['<Super>1']
```

---

Переключение рабочего стола на обоих мониторах

```bash
gsettings set org.gnome.mutter workspaces-only-on-primary false
```
