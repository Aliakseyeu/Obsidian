- В ~/.local/share/applications/ создать файл PhpStorm.desktop с содержимым

```
[Desktop Entry]
Version=1.0
Type=Application
Name=PHPStorm
StartupWMClass=jetbrains-phpstorm
Comment=PHP IDE
Exec=/opt/PhpStorm-241.19072.22/bin/phpstorm.sh
Icon=/opt/PhpStorm-241.19072.22/bin/phpstorm.png
Terminal=false
Categories=Development;IDE;
```

- Открыть свойства и сделать файл [[Исполняемый файл]]
- Запустить приложение
- В терминале ввести xprop WM_CLASS
- Открыть окно приложения и навести на него курсор
- Курсор должен измениться на крестик
- Кликнуть в окне
- В терминале увидеть что-то вроде WM_CLASS(STRING) = "jetbrains-phpstorm", "jetbrains-phpstorm"
- В параметр StartupWMClass установить любое значение из выведенных (jetbrains-phpstorm)
