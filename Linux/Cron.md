Разберём расписание:

`59 8 * * *`

---

### 🔹 Формат cron-записи:

```
 ┌───────── минута (0–59) 
 │ ┌─────── час (0–23)
 │ │ ┌───── день месяца (1–31) 
 │ │ │ ┌─── месяц (1–12) 
 │ │ │ │ ┌─ день недели (0–7, где 0 и 7 = воскресенье) 
 │ │ │ │ │ 
 │ │ │ │ │ 
59 8 * * *
```


---

### 🔹 Расшифровка:

- `59` — 59-я минута,
- `8` — 8 часов утра,
- `* * *` — каждый день, каждый месяц, любой день недели.

---

### ✅ Значит:

> Крон выполняется **каждый день в 08:59 утра** по системному времени сервера.

---

0 7 7 * *
